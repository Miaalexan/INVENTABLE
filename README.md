# INVENTABLE

INVENTABLE - Sistema Web para Gestión de Restaurantes

Requerimientos:
- Python 3.10+
- Django 5+
- SQLite3 (incluido por defecto)

Instalación:
1. Clonar el repositorio:
   git clone https://github.com/Miaalexan/INVENTABLE.git

2. Instalar dependencias:
   pip install -r requirements.txt

3. Ejecutar migraciones:
   python manage.py migrate

4. Ejecutar servidor:
   python manage.py runserver

Estructura del proyecto:
- clientes/      -> gestión de clientes
- usuarios/      -> administración de personal
- reservas/      -> gestión de reservas
- pedidos/       -> creación y seguimiento de pedidos
- menu/          -> catálogo de productos
- reportes/      -> consultas de ventas

Cómo usar:
- Acceder a localhost:8000
- Crear clientes y productos
- Crear reservas
- Registrar pedidos
- Revisar reportes
